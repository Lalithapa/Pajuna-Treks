# Pajuna — Pahad. Jungle. Nadi.

A mindful, intentional trekking community committed to sustainable Himalayan exploration.

## About

Pajuna is a Himalayan trekking community built around the idea that travel should be purposeful — not just scenic. We curate immersive experiences across treks, leisure camping, mountain marathons, and wilderness expeditions, with a strong focus on sustainability, safety, and real human connection.
